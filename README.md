This script scrapes the list of OA journals on the Elsevier website that are part of the TLCUA agreement: [https://agreements.journals.elsevier.com/tlcua?pageNo=1](https://agreements.journals.elsevier.com/tlcua?pageNo=1)

The script was written in Python 3.11.  It uses requests and beautifulsoup for downloading and parsing the data from the website.  The data is saved as an ANSI-encoded CSV file.
